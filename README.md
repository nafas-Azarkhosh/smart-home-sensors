# Smart Home Sensor System 🏠

An object-oriented smart home sensor system created with Python.

## About Project

This project is a simple smart home simulation that manages different types of sensors using object-oriented programming concepts.

## Features

- Create different types of sensors
- Temperature monitoring
- Motion detection
- Humidity monitoring
- Camera status management
- Alert system for temperature limits
- Object-oriented design

## OOP Concepts Used

- Abstraction
- Inheritance
- Encapsulation
- Polymorphism

## Technologies

- Python

## How to Run

Run the project using:

python smart_home_sensor.py
